# Baby Tracker: Дата высадки

Космический трекер беременности на React + Vite.

## Что умеет приложение

- Ввод даты последней менструации (LMP)
- Автоматический расчет срока: недели и дни
- Расчет предполагаемой даты родов (ПДР)
- Отображение текущей недели беременности
- Космическая сцена с плавно анимированным фруктом/овощем недели
- Случайные мотивационные фразы для мамы
- Адаптивный интерфейс для mobile, tablet и desktop

## Технологии

- JavaScript
- React (функциональные компоненты и хуки)
- Vite

## Запуск проекта

```bash
npm install
npm run dev
```

После запуска открой адрес из терминала (обычно `http://localhost:5173`).

## Прод-сборка

```bash
npm run build
npm run preview
```
